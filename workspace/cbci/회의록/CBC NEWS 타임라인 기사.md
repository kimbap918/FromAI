

## 1. Home (홈) 페이지
1. 사용자에게 뉴스 카테고리(총수家, 기업家 등)를 보여줍니다.
2. 총수家에서는 대기업 총수 51명의 실제 프로필 정보를 기반으로 한 목록이 구성되어 있습니다.
3. 개별 목록 클릭 시 해당 총수의 타임라인 기사 페이지로 이동합니다.

![](https://i.imgur.com/WY9BBb1.png)

<br>


## 2. ChairmanDetail (총수 상세) 페이지
1. 선택된 총수의 상세 프로필 정보와 관련된 뉴스 타임라인 이벤트를 보여줍니다.
2. 타임라인의 상단에는 총수의 실제 인적 정보를 표기합니다.(CBC 뉴스 레이아웃과 동일)
3. 관리자 권한이 있는 경우, 타임라인 이벤트를 수정하거나 삭제할 수 있는 기능을 제공합니다.
![](https://i.imgur.com/7PFd6kY.png)


<br>

## 3. AdminNews (뉴스 관리) 페이지
1. 관리자가 시스템에 수집된 원본 뉴스 기사들을 검토하고 관리하는 페이지입니다.
2. 뉴스는 기본적으로 수집된 전체 기사가 목록에 나오나, **기사를 선택하거나 검색 시 개별 기업별 뉴스만 분류되어 기사 목록에 보이게 됩니다.**
3. 특정 총수에 관한 기사들을 선택하여 AI를 통해 타임라인 기사를 생성하고, 미리보기 및 게시할 수 있습니다.
4. 생성된 기사는 수정을 통해 기사를 다듬을 수 있는 기능을 제공합니다.
5. 생성된 타임라인 기사를 게시하면 해당 기사들은 처리 완료(consumed) 상태로 변경되어 목록에 사라집니다.

추후 권재웅 이사님의 뉴스 API 개발 완료 시 연합뉴스의 기사를 수집하여 총수별/기업별로 분리하여 정제 후, 기사를 생성할 수 있도록 수정하겠습니다.
