# 감성 분석 모델 분석 보고서 (Sentiment Model Report)

본 보고서는 CBC News에 도입한 감성 분석 모델의 특성과 하이브리드 시스템에서의 역할을 분석합니다.

<br>

## 1. 모델 개요
- **모델명**: `FISA-conclave/klue-roberta-news-sentiment`
- **기반 아키텍처**: RoBERTa-Small (KLUE base)
- **학습 데이터**: 한국어 뉴스 말뭉치 (KLUE 기반) 및 금융/뉴스 도메인 데이터셋
- **주요 특징**: 일반 영화 리뷰(NSMC)와 달리, 비즈니스 및 사회 뉴스에 등장하는 객관적 서술문 속의 숨은 감성을 파악하는 데 최적화됨.

<br>

## 2. 모델 선정 이유
기존 영화 리뷰용 모델(`nsmc`)은 비즈니스 뉴스의 "실적 개선", "신년사", "비전"과 같은 선언적 문구를 부정(Negative)으로 오판하는 경향이 있었으나, 
`klue-roberta-news-sentiment`는 다음과 같은 강점을 가짐
- **도메인 적합성**: 뉴스 문체에 익숙하여 "하락세를 극복"과 같은 복합 문맥을 정확히 파악.
- **경량성**: RoBERTa-Small 기반으로 CPU 환경에서도 빠른 추론 속도를 보장.

<br>

## 3. 하이브리드 시스템 내 역할
본 모델은 단독으로 결정권을 가지지 않고, 사용자 정의 **가중치 키워드 사전**과 결합되어 동작.

<br>

### A. AI (Contextual Awareness)
- 문장의 전체적인 어조(Tone)와 맥락을 파악.
- 키워드가 없는 문장에서도 전체적인 흐름을 통해 `Positive` / `Negative` 신호 추출.

<br>

### B. Keywords (Domain Anchoring)
- `수사`, `횡령` (강력 부정) / `MOU`, `흑자` (강력 긍정) 등 뉴스의 핵심 사실을 기반으로 AI의 판단을 보정.
- AI가 0.75 이하의 낮은 확신도를 보일 때, 키워드 가중치가 결정적인 판정 방향을 제시.

<br>

## 4. 분석 성능 최적화 (Optimization)
- **Synergy Boost**: AI 판단과 키워드 방향이 일치할 때 Confidence 점수를 **최대 15%** 상향.
- **Veto Mechanism**: '가짜 호재' 방지를 위해 치명적 악재 키워드 감지 시 AI 긍정 신호를 중화.
- **Speech Anchor**: '신년사', '비전' 기사가 AI에 의해 공격적으로 분류되는 것을 방지하기 위해 특화 보너스 점수 부여.

<br>

## 5. 향후 개선 방향
- **파인튜닝(Fine-tuning)**: 장기적으로 수집된 CSV 데이터를 바탕으로, 당사 도메인(재계/회장님 관련)에 특화된 추가 학습 가능.
- **엔티티 인식(NER) 연동**: 감성 분석 결과가 특정 인물(`PERSON`)이나 기업(`Group`)에 대한 것인지 더 정교하게 연결.
