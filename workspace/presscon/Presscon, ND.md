## Presscon에만 있는 기능

### 1. 대시보드

1. SEO 진단(PC/모바일) 대시보드 위젯
	- **PC SEO 점수 / Mobile SEO 점수**를 분리해서  
	**성능/접근성/권장사항/검색엔진 최적화**를 원형 지표로 보여줌
![](https://i.imgur.com/H8jLan8.png)

2. 광고 매출 대시보드(광고 지표가 별도 존재)
- **광고 매출** 차트가 있고, 범례에 **노출 / CLICK+ / 광고수익**이 있음  
    → ND 쪽 스샷에는 광고 매출 관련 위젯이 보이지 않음
![](https://i.imgur.com/0NpMObg.png)


3. 기자 활동 내역(기자별 생산성/성과 테이블)
- 대시보드에 **기자 활동 내역** 섹션이 있고, **출판 기사 합계 / 일 평균 출판기사 / 평균 PV / 평균 UV** 같은 기자별 랭킹 테이블 존재
![](https://i.imgur.com/odr9e5u.png)


<br>

### 2. 기사 리스트
1. 콘텐츠 관리(CMS)에 가까운 기사 리스트 구성
- 상단 필터에 **기간 + 1차 섹션/2차 섹션 + 등급 + 검색어**(드롭다운/인풋) 구성
* 리스트가 더 편집/발행 중심:
    - **대표이미지, 네트워크 송출, 출판일, 수정일, 조회수, 편집, 상태, 포털송출**
    - 각 행에 **웹출판 / 송출확인** 같은 버튼이 존재
- 상단 액션에 **섹션 변경 / 기사 삭제 / 출판 최신순** 같은 운영 버튼
- 우측 상단에 **기사 보관함 / 기사댓글관리** 버튼이 따로 있음
![](https://i.imgur.com/eMAsgXS.png)



2. 기사 작성 시 이미지 모듈(예정)
	- 업스케일러, 사진필터, 다양한 확장자 지원을 통한 사진 편집 최적화 기능 


<br>

### 3. 화면 관리
1. 화면 관리와 동시에 페이지 관리 가능
	- ND의 부가기능에 있는 페이지 관리 기능을 화면 관리에 넣음으로서 화면 관리와 페이지 관리를 동시에 가능

2. 다양한 언어 지원 
	- 페이지 내의 언어를 다양하게 지원함으로서 다국적 서비스에 유리

![](https://i.imgur.com/VZDGQOk.png)


3. 화면에서 바로 확인 가능한 광고 배너
	- 광고 위젯 설정 시 바로 편집 가능한 광고
	- 저장 및 적용시 실시간으로 적용 가능한 광고 

![](https://i.imgur.com/bF9lo5I.png)

<br>

### 4. 멤버 관리

1. 멤버 관리의 편의성
	- 쉬운 권한 관리, 삭제, 추가 기능 

![](https://i.imgur.com/xbxz1id.png)

## ND에만 있는 기능

### 1. 대시보드
1. 운영 큐(Queue) 모니터링에 가까운 구성
	- 큰 카드 운영 상태 숫자가 큼직하게 존재해 매우 직관적
	- 대시보드 하단에 **미승인/예약/최근등록/댓글** 같은 처리해야 할 리스트 위젯이 바로 노출

![](https://i.imgur.com/NLXwVOv.png)


<br>


### 2. 작성
1. 작성 편의성
	- 기사 작성 시 한 눈에 들어오는 편집기
	- 작성 후 저장 시 최소화된 동선

![](https://i.imgur.com/8jQaH4h.png)



### 3. 승인
1. “승인” 중심의 운영 플로우가 더 전면에 위치
	- 기사 리스트가 ‘승인’ 화면으로 구성, 상단 탭이 전체승인 / 미승인 / 예약승인 / 색인·송고삭제 / 포털전송 으로 나뉨
	- 체크박스 선택 후 일괄 승인 버튼이 바로 있음

![](https://i.imgur.com/8QfGkAP.png)

2. 포털 전송 상태/대상 표기가 매우 직관적
	- 리스트에서 각 기사별로 포털 아이콘(N, D, Z, R) 이 컬러 배지로 표시됨  
	    → “어디로 나가냐/나갔냐”가 한 눈에 보임
	- 우측에 **‘취소’** 버튼이 각 행에 존재(승인/송출 취소 성격으로 보임)

![](https://i.imgur.com/OXySWYl.png)

3. 빠른 속도와 편의성
	- 기사 승인을 위한 기사 클릭 시 화면을 분할하여 곧바로 나타나는 기사 창
	- 승인 또한 비동기 형태로 이뤄져 페이지의 새로고침 없이 빠른 처리 및 확인 가능

<br>

	- 편집을 위한 화면이 단순히 예시화면이 아닌, 현재 송출중인 기사를 보여줌
	- 이를 통해 화면 저장 시 곧바로 변경될 화면을 알수있음 

![](https://i.imgur.com/p8mmvjX.jpeg)

<br>

2. 컴포넌트의 배치 편의성

![](https://i.imgur.com/Pzffrdz.png)


<br>

3. 상세한 화면 편집 지원
	- Presscon이 메인 편집과 푸터 관리만 있는것에 비해 
	- 메인/모바일/기사/기사 리스트 로 다양한 화면 편집 지원

![](https://i.imgur.com/UekODOe.png)
